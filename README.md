# KeepOrSkip

KeepOrSkip is a simple web tool that helps you quickly decide whether something is worth it or should be skipped.

## How it works

Enter an option or idea, and the tool evaluates it to give you a clear **keep** or **skip** result with a short explanation.

## Features

- Fast decision making
- Clean and minimal interface
- Simple keep/skip logic
- No distractions

To reduce decision fatigue and help users make faster, clearer choices.
